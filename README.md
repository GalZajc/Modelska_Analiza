# Modelska analiza 1 in 2

Ta repozitorij vsebuje urejena koncna porocila in izbrane MP4 animacije za predmeta Modelska analiza 1 in Modelska analiza 2.

Vkljucene so:
- vse ostevilcene mape nalog iz `ModAn1`
- `ModAn1 Zakljucna`
- vse ostevilcene mape nalog iz `ModAn2`

V vsaki mapi so objavljena samo:
- navodila v obliki `mod...pdf`
- moje porocilo v obliki `GalZajc...pdf`
- datoteke `mp4`, ce so prisotne v tej mapi
